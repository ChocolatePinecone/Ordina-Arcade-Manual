To play, make sure the arcade cabinet is plugged in and the red power switch on the back is turned on.

If needed you can adjust the audio using the knobs at the top right above the screen.

When playing actual "Arcade" games you need to use the "coin" and 1 or 2 player buttons at the front to start the game.

Info

This cabinet was made as as fun side project by Jelmer Pijnappel.
The cabinet casing and parts were provided by Lumimaniac, while the cabinet design and electric connections were done by Jelmer.

Internally the cabinet is built using a raspberry pi, 2 arcade interface boards for the controllers, an equalizer and amplifier board + 2 speakers for the audio and a led strip for the lighting.
